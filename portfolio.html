Projeto sobre mim

Este projeto foi desenvolvido durante os cursos da Alura na formação de HTML e CSS. Trata-se de um site pessoal estático, construído com o objetivo de aplicar na prática os conhecimentos fundamentais de estruturação de páginas com HTML5 e estilização com CSS3.

Durante o desenvolvimento, foram abordados conceitos como organização de conteúdo em múltiplas páginas (home, sobre e currículo), boas práticas de estrutura de pastas (assets, styles) e aplicação de estilos com uso de classes, cores, fontes e posicionamento com Flexbox. O projeto também reforça a importância da separação entre estrutura (HTML) e apresentação (CSS).

O foco principal foi criar uma página de apresentação pessoal, simulando um pequeno portfólio online, utilizando apenas tecnologias front-end básicas. Esse tipo de projeto é ideal para quem está começando na área de desenvolvimento web e serve como base para evoluir para páginas mais dinâmicas e interativas.

Mobile fist - AluraBooks

Este projeto foi desenvolvido durante o curso da Alura com foco na técnica de Mobile First, utilizando apenas HTML e CSS. O objetivo principal é criar um site responsivo, priorizando a experiência em dispositivos móveis e, em seguida, adaptando o layout para telas maiores com media queries. O projeto serve como prática para aplicar conceitos de responsividade, organização de conteúdo e boas práticas de desenvolvimento front-end moderno.

Projeto Uninter (em desenvolvimento)

Este projeto "Sobre Mim" foi desenvolvido como atividade prática para a Uninter, utilizando exclusivamente HTML e CSS. Sua base vem do meu primeiro projeto da formação em Front-end na plataforma Alura, por isso possui uma estrutura semelhante. A proposta foi reaproveitar esse modelo inicial e adaptá-lo, aplicando os conhecimentos adquiridos ao longo dos estudos, como responsividade com Mobile First, organização de pastas e arquivos, estilização com Flexbox e boas práticas de desenvolvimento front-end. Mesmo sendo uma atividade acadêmica, o projeto também contribui para meu portfólio pessoal e demonstra a evolução prática obtida com o suporte dos cursos complementares da Alura.
